* Styling and Design Highlights

Responsive Layout:
The website adapts fluidly across devices using CSS Grid and flexible units.

Color Themes:

Light Mode: Soft lavender and white tones for clarity and simplicity.

Dark Mode: Deep violet and royal purple palette for a sleek, modern aesthetic.

Typography:
Uses the Poppins font for a clean, geometric, and professional feel.

Interactive Elements:
Buttons and links include smooth hover transitions and shadow effects.
Cards, panels, and tables have soft depth shadows and rounded corners for visual consistency.

Skill Section:
Animated skill progress bars visually represent proficiency using intersection observers for scroll-based activation.

Contact Form:
Includes client-side validation for name, email, and message fields, providing error feedback and a success message on correct input.

* JavaScript Functionality

Theme Switcher (Home Page Only)
Toggles between light and dark themes.
Saves user preference in localStorage for future visits.

Form Validation (Contact Page)
Checks required fields and validates email format.
Displays inline error messages or success feedback.

Animated Skill Bars (About Page)
Triggers skill bar animations when the section comes into view.
Uses Intersection Observer for performance-friendly activation.

* Key Features

 Fully responsive (mobile, tablet, and desktop)
 Dual light/dark themes with smooth transition
 Animated skill bars and hover effects
 Validated contact form (demo mode)
 Professional, minimalistic UI/UX design
